個人近況 now

( ?  read )
`/now` 是本部落格的近況頁面，用來記錄那些當朋友問說「嘿，最近在做什麼呀？」時會回答的內容。源自 Derek Sivers 發起的 now page 運動（ 1, 2, 3）
有興趣的朋友可以在 nownownow.com/TW 逛逛各個部落格的 `/now` 頁面，看看大家的近況
或是也可以在自己家捏一個 `/now` 並投稿（1, 2）來一起參與 nownownow 喲！

更新時間：2026-06-21

最近的狀態

在看了幾篇其他部落格朋朋介紹 nownownow.com 的文章後，產生了「也想要讓這個部落格也出現在 nownownow.com 吶。」的想法，於是動手弄了一個 now 頁面。

但還在思考貼文（post）、動態（timeline）、近況（now）各自的定位，也在摸索 now 頁面該放的內容……總之先用分段條列式的，無腦又簡單。

個人近況：這陣子動力持續低迷，部落格文章和個人筆記都緩了下來。即使同事分享了許多小工具，心裡也暗暗決定要寫成文章，但丟進草稿區後，三個禮拜就過去了。據說走出迷霧的訣竅在於累積小小的勝利，祝福我會在這些新的嘗試中武運昌隆吧。

最近看的

《極限返航》：趁著前陣子的電影熱潮又重新看了一次小說，非常喜歡主角們的互動。科幻設定、劇情推進跟幽默吐槽之間的節奏很舒服，趁著假日一口氣看完，通體舒暢。

《沉思錄》：剛開始讀的時候，心想這人「宇宙」「本性」嘰哩咕嚕的都在說些什麼呀，結果讀完之後意外地還不錯，像是聽了一位朋友在碎碎念一樣，順手收了一堆喜歡的格言到收藏庫裡，謝謝馬可。

最近玩的

在「我必須有一段完整的時間，才能好好享受這些作品」的執念下，《光與影：33 號遠征隊》和《活俠傳》等大型作品的進度停滯不前。反而是每場不用半小時的肉鴿遊戲和《爐石戰記》英雄戰場的時間占比急速提升中。

灰燼之國 Cinderia：搶先體驗中的 Roguelite 動作遊戲，和 Hades 一樣是俯視角砍怪和抽技能搭配，但因為技能組合夠多、剛入坑的新鮮感也還在，玩起來還蠻紓壓的。

艾爾登法環：五月底的時候回鍋了法環。漫無目的地在寧姆格福逛逛街、探探地牢，還蠻悠閒的。對比前面提到的「想要留時間好好享受新遊戲」的心魔，已經通關的遊戲反而能沒有壓力地開來玩，總覺得這樣的心態也有點微妙。

最近聽的

Gotye - Somebody That I Used To Know (feat. Kimbra)

前陣子在社群媒體被凸槌拍檔的版本洗腦（意外地有種魔性，聽了十幾次之後整個腦袋都是那個旋律）


乾脆去找了原唱來聽，結果被 MV 的內容所吸引。那些色塊到底代表著什麼呢？
