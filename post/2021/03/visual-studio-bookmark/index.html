Visual Studio: 書籤 (bookmarks)

2021-3-13,
Tools IDE
( ?  read )

今天從同事們那邊學到了書籤這個方便功能，趁還記得的時候來做個紀錄。

那麼馬上就來操作一次：

使用 Ctrl K, Ctrl K 可以在指定的行號上加上一個「書籤」

接著我們就能在書籤視窗中看到這個書籤（檢視＞書籤視窗，或是 Ctrl K, Ctrl W）

在書籤視窗中，也可以對書籤重新命名，或是建立資料夾進行管理

而當頁面中有多個書籤的時候，就可以利用 Ctrl K, Ctrl P 和 Ctrl K, Ctrl N 來在書籤中移動

這個快速移動能用在什麼時候呢？

我們在進行一個專案時，會常常需要在多個檔案，或是在一個較長的檔案的各處來來回回。

例如 private 方法離使用到的 public 方法有點遠，又或是我個人就常常會寫某個 Function 到一半的時候，突然發現：「啊！我忘記在建構式的時候把要用的對象傳進來了。」

這時候就需要來回跑確認方法內容，或是移動到整個類別的頂部去增加宣告和修改建構式，接著再回到剛剛撰寫中的方法，在程式碼裡面折返跑，好不愉快。

以往遇到這種狀況，我會稍微記一下行號，再利用 Ctrl G 跳回去。

今天跟同事聊到書籤的時候，就想到：像是上面的場景，便可以利用書籤來紀錄常用的幾個地方，並快速切換。並且用法也簡單直覺。大概就像飛雷神之術一樣吧。這邊就記錄一下，也分享給大家。
