聊一下外接螢幕

2020-10-11,
Tools w3HexSchool
( ?  read )

這週入手了新玩具，在這邊記錄下～

我們家大神大大跟我說部落格就當推特發就對了，唉呀我也是深表認同吶

像我這種被實驗室和公司寵壞的人，已經習慣了雙螢幕的好。結果不管是在放不下兩台螢幕的家裡書桌，還是帶著筆電出門，尤其是一邊寫東西一邊查資料，需要來來回回切換視窗時，總是會想「唉呀真想把這丟到另一個螢幕啊！」

因此！幾經掙扎之後，還是入手了外接螢幕！

開場先說心得，雖然尚未有外出機會，但目前的使用相當滿意。這邊就說說個人體會的好壞：

2021/1/30 補充：外出也相當方便，只是桌面的空間就需要大一點

外接螢幕的讚

  • 有雙螢幕很爽
  • 可以攜帶出門，到圖書館之類的地方也能有雙螢幕很爽
  • 雙螢幕在一邊 Coding 一邊 抄Stackoverflow 查資料，或是像部落格一邊編輯一邊看結果的時候很爽

其實整個說起來就是帶著筆電也能雙螢幕的那種爽感，一旦用過雙螢幕，感受到不用一直切換視窗的流暢感之後，就真的回不去啦。

外接螢幕的沒那麼讚

  • 加上保護套之後幾乎都要一公斤左右，不太能忽視
  • 出門在外還要接一堆線有時候不太美觀
  • 背包裏面又是筆電又是螢幕又是手機錢包的，身家財產哪！包包離開視線就會心神不寧

誰適合用外接螢幕

  • 有出外工作習慣，例如跑去咖啡廳帥一天的朋友們
  • 像我一樣桌電筆電交替使用的朋友們
  • 只是想要出門用大螢幕打 Switch 的朋友們

大概先這樣，推特也有字數限制呢。等有外出機會還是有什麼心得再來補充一下，先去享受新玩具了 這週也是今年最後一個連假了，各位開工愉快，咱們下周見

補充一下：我個人入手的型號是 MSI Optix MAG161V，算是相當輕便好看。線材接的位置也挺不錯的，螢幕看著也挺舒服。

Google 一下也有挺多開箱文和介紹影片的，有興趣可以看看。我這真拍不出像樣的開箱文，太難了，我嘗試過了…。這邊就推薦一下外接螢幕這東西，有興趣的朋友再自己做功課吧～
