DevToys —— 開發人員的瑞士刀工具箱

2022-8-20,
Tools
( ?  read )

嗨各位朋朋，又双叒叕到了「同事推薦的好用工具」時間！

今天要介紹的是 Devtoys：這是一套包含了許多貼心小工具的工具箱，例如 UUID 產生器、Base64 編碼解碼、JSON Format 都可以在這邊找到，省下 Google 這些小工具的時間。

進入官網或 Store 下載之後，點開就能在畫面上看到一整堆小工具：

左邊的工具列也有工具分類和搜尋可以使用，具體有哪些工具這邊就不再贅述。可以到 Github 上看一下工具列表，或是就直接下載下來看看更清楚：

例如說當我原本複製了一坨 Json，為了保護眼睛和心靈，我就得上網先找個 Json Formatter 來轉一下；現在我知道 Devtoys 就有，直接打開來用就可以。

如果真的很常用到的工具，也可以加到最愛，就會直接出現在左上角的列表中：

同時 DevToys 也支援釘選到 Windows 的「開始」功能表，如果真的很常用也可以直接釘起來：

其他遇到的場景還有：複製了 JWT Token 想快速解開、想從 Unix 時間戳轉出時間等等

例如我個人最近常跟排程類的服務打交道，就會需要時常調整 Crontab 語法，而 DevToys 當然也有提供工具來幫助我們迅速確認 Cron 的執行時間。

像這種時候只要先確認一下 DevToys，也許就能找到對應的工具，下次再遇到就可以省下找工具的時間，可謂是方便方便。

那麼這次的小工具分享就到這邊，再度成功一個工具水一篇文，推薦大家下載試試！
